## Analysis Framework

For each site, evaluate:

### Visual Design
- Design system and consistency
- Colour usage and palette
- Typography (system fonts vs. custom)
- Iconography and illustration
- Photography style (stock vs. NZ-specific)
- Visual hierarchy

### Layout & Structure
- Information architecture patterns
- Navigation models
- Search functionality
- Content page templates
- Landing page designs
- Footer and utility navigation

### User Experience
- Task-oriented design
- Plain language usage
- Progressive disclosure of complex information
- Form design and user flows
- Error handling
- Responsive design approach

### Accessibility
- WCAG compliance level
- Screen reader compatibility
- Keyboard navigation
- Colour contrast
- Text sizing and readability
- Alternative text usage
- Focus indicators

### Bilingual Design (English / Te Reo Māori)
- Language switching mechanism
- Te Reo integration approach
- Macron handling
- Bilingual navigation
- Content parity between languages
- Cultural design elements (Māori design motifs)

### Content Strategy
- Content types and templates
- Plain language approach
- Data presentation and visualisation
- Document/publication management
- News and updates
- Consultation and engagement features

### Technical
- Platform and CMS
- Performance and page speed
- Progressive enhancement
- Mobile-first approach
- Search engine optimisation
- Analytics integration

### Relevance to Alliance
- Accessibility standards to adopt
- Bilingual design patterns for Te Reo integration
- Trust and credibility design patterns
- Information architecture models
- NZ-specific design conventions users expect
