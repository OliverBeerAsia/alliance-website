## Analysis Framework

For each site, evaluate:

### Visual Design
- Colour palette and usage
- Typography choices (headings, body, accent)
- Photography/imagery style
- Use of illustration or iconography
- White space and density
- Overall visual tone (bold/subtle, warm/cool, energetic/calm)

### Layout & Structure
- Homepage structure and sections
- Navigation pattern (sticky, hamburger, mega-menu)
- Hero section approach
- Grid system and content blocks
- Footer design and content
- Page template variety

### User Experience
- First impression and emotional impact
- Information hierarchy
- Call-to-action placement and design
- Mobile responsiveness
- Page load performance
- Scroll behaviour and animations

### Content Strategy
- Content types present (blog, policy, news, stories, media)
- Storytelling approach
- Use of video and multimedia
- Social proof elements
- Supporter/member stories
- Email capture strategy

### Conversion & Engagement
- Primary CTAs (donate, join, volunteer, sign petition)
- Donation page design
- Volunteer sign-up flow
- Email list building approach
- Social media integration
- Event promotion

### SEO & Technical
- URL structure
- Meta descriptions approach
- Heading hierarchy
- Schema markup usage
- Site speed (if testable)
- Accessibility features

### WordPress Relevance
- Is the site built on WordPress?
- Identifiable themes or page builders
- Plugin patterns worth noting
- Performance approaches transferable to WordPress

### Relevance to Alliance
- What could be adapted for NZ context
- What doesn't translate (cultural/scale differences)
- Specific elements to consider adopting
