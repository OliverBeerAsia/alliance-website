## Analysis Framework

For each site, evaluate:

### Visual Design
- Colour palette and brand consistency
- Typography system
- Photography and imagery approach
- Use of data visualisation
- Visual identity strength
- Cultural design sensibilities

### Layout & Structure
- Homepage structure
- Navigation complexity and depth
- Policy page architecture
- News/blog integration
- Campaign landing pages
- Multilingual handling

### User Experience
- Information density vs. clarity
- Policy content accessibility
- Member vs. public content
- Mobile experience
- Accessibility compliance
- Language switching (if multilingual)

### Content Strategy
- Policy communication depth
- Use of manifestos and programme documents
- News and media sections
- Candidate/representative profiles
- Local branch/chapter pages
- Historical/archive content

### Conversion & Engagement
- Membership sign-up flow
- Donation mechanisms
- Volunteer pathways
- Event listings and registration
- Newsletter integration
- Social media presence

### SEO & Technical
- URL structure (especially multilingual)
- Content management approach
- Performance
- Accessibility standards
- Progressive web app features

### WordPress Relevance
- Platform/CMS used
- Design patterns transferable to WordPress
- Content management approaches applicable to WordPress
- Plugin equivalents for key features

### Relevance to Alliance
- Policy communication models to adopt
- Membership engagement approaches
- Bilingual/bicultural design lessons (for English/Te Reo)
- Party structure communication (democratic socialist parties)
- Coalition/movement-building messaging
